# 2-07 HTTP相关
## HTTP简介
### 超文本传输协议HTTP主要特点

- 支持客户/服务器模式
- 简单快速
- 灵活：content-type
- 无连接：每次连接处理一个请求，1.1开始默认使用长连接keepAlive
- 无状态：协议对于事务处理没有状态，需要前面的信息需要重传
