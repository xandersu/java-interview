## 说说tcp三次握手
### 传输控制协议TCP简介：

- 面向连接的、可靠的、基于字节流的传输层通信协议
- 将应用层的数据流分割成报文段并发送给目标节点的TCP层
- 数据包都有序号，对方收到则发送ACK确认，未收到则重传
- 使用校验和来校验数据在传输过程中是否有误

### TCP Flags

- URG：紧急指针标志
- ACK：确认序号标志
- PSH：push标志
- RST：重置连接标志
- SYN：同步序号，用于建立连接过程
- FIN：finish标志，用于释放标志
